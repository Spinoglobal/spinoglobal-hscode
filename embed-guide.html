🔧 Hướng dẫn nhúng công cụ HS Code vào website

Sau khi deploy lên GitHub Pages, bạn có thể nhúng công cụ vào bất kỳ website nào.

1. Nhúng cơ bản (iframe)

<iframe 
  src="https://YOUR_USERNAME.github.io/spinoglobal-hscode/" 
  width="100%" 
  height="800" 
  frameborder="0"
  style="border:none"
></iframe>

2. Nhúng đẹp hơn (có shadow + bo góc)

<div style="max-width:1000px;margin:20px auto">
  <iframe 
    src="https://YOUR_USERNAME.github.io/spinoglobal-hscode/" 
    width="100%" 
    height="850" 
    frameborder="0"
    style="border:none;border-radius:12px;box-shadow:0 4px 20px rgba(0,0,0,0.1)"
  ></iframe>
</div>

3. WordPress

Thêm block Custom HTML rồi paste đoạn iframe ở trên.

4. Nhúng full-page (toàn trang)

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Tra cứu HS Code - Spino Global</title>
  <style>
    body{margin:0;padding:0;overflow:hidden}
    iframe{width:100%;height:100vh;border:none}
  </style>
</head>
<body>
  <iframe src="https://YOUR_USERNAME.github.io/spinoglobal-hscode/"></iframe>
</body>
</html>
⚠️ Lưu ý: Thay YOUR_USERNAME bằng tên GitHub của bạn. Nếu đặt tên repo khác thì thay spinoglobal-hscode tương ứng.

5. Tùy chỉnh chiều cao tự động

<script>
window.addEventListener('message', function(e) {
  if (e.data && e.data.type === 'resize') {
    document.getElementById('hscode').style.height = e.data.height + 'px';
  }
});
</script>
<iframe id="hscode" src="https://YOUR_USERNAME.github.io/spinoglobal-hscode/" 
  width="100%" height="800" frameborder="0" style="border:none"></iframe>
